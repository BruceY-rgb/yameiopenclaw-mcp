# 腾云商旅 MCP Server 使用指南

## 快速开始

### 1. 配置环境变量

在项目根目录创建 `.env` 文件：

```bash
# 公司密钥（从运营后台获取）
ONTUOTU_APP_KEY=你的AppKey
ONTUOTU_APP_SECRET=你的AppSecret

# MCP 模式
MCP_TRANSPORT=stdio   # Claude Desktop 用 stdio
# MCP_TRANSPORT=http  # 远程调用用 http
# MCP_PORT=8000       # HTTP 模式端口
```

### 2. 启动 MCP Server

```bash
# 安装依赖
pip install -r requirements.txt

# 启动（stdio 模式，默认）
python -m src.server

# 或 HTTP 模式
MCP_TRANSPORT=http python -m src.server
```

### 3. 在 Claude Desktop 中使用

在 `claude_desktop_config.json` 中添加配置：

```json
{
  "mcpServers": {
    "openclaw": {
      "command": "python",
      "args": ["-m", "src.server"],
      "env": {
        "ONTUOTU_APP_KEY": "你的AppKey",
        "ONTUOTU_APP_SECRET": "你的AppSecret"
      }
    }
  }
}
```

重启 Claude Desktop 后，即可使用 MCP 工具。

| 工具名 | 功能 | 示例 |
|-------|------|------|
| `create_user_account` | 创建用户账号 | 创建新的出行人账号 |

## 完整操作流程

### 场景：预订一张上海到伦敦的机票

#### 步骤 1：创建用户账号（首次）

```
用户：帮我创建一个用户账号
Claude：调用 create_user_account 工具
结果：创建成功，返回用户名和密码
```

#### 步骤 2：查询航班

```
用户：查询上海到伦敦的航班，2026年4月15日出发
Claude：调用 search_international_flights 工具
参数：
- from_city: SHA（上海浦东机场）
- to_city: LHR（伦敦希思罗机场）
- departure_date: 2026-04-15
- adult_count: 1

结果：返回航班列表，包含价格、航司、起飞时间等信息
```

#### 步骤 3：创建出行人

```
用户：添加一个出行人，张三，护照号 E12345678
Claude：调用 create_passenger 工具
参数：
- name: 张三
- passenger_type: ADT（成人）
- id_type: passport
- id_number: E12345678
- nationality: CN（中国）
- gender: male
- birthday: 1990-01-01

结果：返回出行人ID
```

#### 步骤 4：预订机票

```
用户：预订第一个航班
Claude：调用 book_international_flight 工具
参数：
- flight_id:（从步骤2获取）
- passenger_id:（从步骤3获取）
- contact_name: 张三
- contact_phone: 13800138000
- contact_email: zhangsan@example.com

结果：返回订单号
```

#### 步骤 5：查询/支付订单

```
用户：查看订单状态
Claude：调用 query_order 工具
参数：
- order_id:（从步骤4获取）

结果：返回订单详情，包含价格、状态等
```

## 工具参数说明

### create_user_account

```
无参数（使用环境变量中的 AppKey/AppSecret）

返回：
- username: 用户账号
- password: 用户密码
```

### search_international_flights

| 参数 | 说明 | 示例 |
|-----|------|------|
| from_city | 出发城市代码 | SHA, PVG, PEK |
| to_city | 目的城市代码 | LHR, JFK, NRT |
| departure_date | 出发日期 | 2026-04-15 |
| return_date | 返程日期（往返） | 2026-04-20 |
| adult_count | 成人数量 | 1 |
| child_count | 儿童数量 | 0 |
| cabin | 舱位 | Y（经济）, C（商务）, F（头等） |

### create_passenger

| 参数 | 说明 | 示例 |
|-----|------|------|
| name | 姓名 | 张三 |
| passenger_type | 类型 | ADT（成人）, CHD（儿童）, INF（婴儿） |
| id_type | 证件类型 | passport（护照）, id_card（身份证） |
| id_number | 证件号码 | E12345678 |
| nationality | 国籍 | CN（中国）, US（美国） |
| gender | 性别 | male, female |
| birthday | 出生日期 | 1990-01-01 |

### book_international_flight

| 参数 | 说明 | 示例 |
|-----|------|------|
| flight_id | 航班ID | （从搜索结果获取） |
| passenger_id | 出行人ID | （从出行人创建获取） |
| contact_name | 联系人 | 张三 |
| contact_phone | 联系电话 | 13800138000 |
| contact_email | 联系邮箱 | test@example.com |

## 常见问题

### Q1: 工具返回 "验签异常"
**解决**：检查 AppKey/AppSecret 是否正确

### Q2: 工具返回 "城市编码有误"
**解决**：使用正确的 IATA 机场代码（如 PVG 而非上海）

### Q3: Claude Desktop 连接不上 MCP
**解决**：
1. 确认已配置 claude_desktop_config.json
2. 检查 python 命令是否在 PATH 中
3. 查看 MCP Server 日志排查
