# 腾云商旅 MCP Server 使用指南

## 快速开始

### 1. 配置环境变量

在项目根目录创建 `.env` 文件：

```bash
# 公司密钥（必填，从运营后台获取）
ONTUOTU_APP_KEY=你的AppKey
ONTUOTU_APP_SECRET=你的AppSecret

# 用户账号（可选，设置后启动时自动登录）
ONTUOTU_USERNAME=你的用户名
ONTUOTU_PASSWORD=你的密码

# MCP 模式
MCP_TRANSPORT=stdio   # Claude Desktop 用 stdio
# MCP_TRANSPORT=http  # 远程调用用 http
# MCP_PORT=8000       # HTTP 模式端口
```

### 2. 启动 MCP Server

```bash
# 安装依赖（使用 uv）
uv sync

# 启动（stdio 模式，默认）
uv run python -m src.server

# 或 HTTP 模式
MCP_TRANSPORT=http uv run python -m src.server
```

### 3. 在 Claude Desktop 中使用

在 `claude_desktop_config.json` 中添加配置：

```json
{
  "mcpServers": {
    "openclaw": {
      "command": "python",
      "args": ["-m", "src.server"],
      "env": {
        "ONTUOTU_APP_KEY": "你的AppKey",
        "ONTUOTU_APP_SECRET": "你的AppSecret"
      }
    }
  }
}
```

重启 Claude Desktop 后，即可使用 MCP 工具。

## MCP 工具列表

| 工具名 | 功能 |
|-------|------|
| `create_user_account` | 创建用户账号 |
| `login_user` | 用户登录 |
| `search_international_flights` | 查询国际航班 |
| `list_passengers` | 查询出行人列表 |
| `create_passenger` | 创建出行人 |
| `book_international_flight` | 预订机票 |
| `query_order` | 查询订单 |
| `pay_order` | 支付订单 |
| `cancel_order` | 取消订单 |

## 工具参数说明

### create_user_account

```json
{
  "real_name": "张三",
  "phone": "13800138000",
  "email": "test@example.com"
}
```

返回自动生成的用户名和密码。

---

### login_user

```json
{
  "username": "用户名",
  "password": "密码"
}
```

---

### search_international_flights

| 参数 | 说明 | 示例 |
|-----|------|------|
| from_city | 出发机场代码 | PVG, PEK, SHA |
| to_city | 目的机场代码 | JFK, LHR, NRT |
| from_date | 出发日期 | 2026-04-15 |
| trip_type | 行程类型 | 1=单程, 2=往返 |
| return_date | 返程日期（往返） | 2026-04-20 |
| adult_count | 成人数量 | 1 |
| child_count | 儿童数量 | 0 |
| infant_count | 婴儿数量 | 0 |
| cabin | 舱位 | Y=经济, C=商务, F=头等 |

返回：航班列表（含 flightID、serialNumber、cacheExpirTime、flightDetailList）

---

### create_passenger

| 参数 | 说明 | 示例 |
|-----|------|------|
| name | 姓名 | 张三 |
| passenger_type | 乘客类型 | 0=成人, 1=儿童, 2=婴儿 |
| nationality | 国籍二字码 | CN, US |
| id_type | 证件类型 | 0=护照 |
| id_number | 证件号码 | E12345678 |
| id_expiration | 证件有效期 | 2030-01-01 |
| gender | 性别 | 1=男, 0=女 |
| birthday | 出生日期 | 1990-01-01 |
| phone | 手机号 | 13800138000 |
| email | 邮箱 | test@example.com |

返回：passenger_id

---

### list_passengers

无参数，返回所有出行人列表。

---

### book_international_flight

| 参数 | 说明 |
|-----|------|
| flight_id | 航班ID（来自搜索结果） |
| serial_number | 序列号（来自搜索结果） |
| passenger_ids | 出行人ID列表 |
| contact_name | 联系人姓名 |
| contact_phone | 联系人电话 |
| contact_email | 联系人邮箱（可选） |
| cache_expir_time | 缓存过期时间（来自搜索结果） |
| flight_data | 完整航班数据（来自搜索结果） |
| search_params | 航班查询参数（来自搜索结果） |
| passenger_infos | 出行人详细信息列表 |

返回：order_id

## 完整操作流程

```
1. create_user_account()     → 创建用户账号（可选，已有账号可跳过）
2. login_user()              → 登录获取 Token
3. search_international_flights() → 查询航班
4. create_passenger()        → 创建出行人（可多个）
5. book_international_flight()   → 下单
```

## 常见问题

### Q1: 工具返回 "验签异常"
**解决**：检查 AppKey/AppSecret 是否正确

### Q2: 工具返回 "城市编码有误"
**解决**：使用正确的 IATA 机场代码（如 PVG 而非上海）

### Q3: Claude Desktop 连接不上 MCP
**解决**：
1. 确认已配置 claude_desktop_config.json
2. 检查 python 命令是否在 PATH 中
3. 查看 MCP Server 日志排查
